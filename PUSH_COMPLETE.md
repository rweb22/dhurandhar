## 🚀 Working Installation Methods

### Method 1: Clone and Install (Most Reliable)

```bash
git clone https://github.com/rweb22/dhurandhar.git
cd dhurandhar
npm install
npm run install:local
```

### Method 2: One-Line Install (Now Available!)

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/rweb22/dhurandhar/main/install.sh)
```
